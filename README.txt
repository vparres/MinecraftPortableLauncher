##################################################
## Portable Minecraft Launcher 1.5.2 by VPARRES ##
##################################################
README Carefully !

This launcher can open Minecraft into an USB Key without any
rights on the PC

the first version is Windows Only

Requirements :

- 100 MB on USB Key
- an Premium Minecraft Account
- An USB Key
- .Net Framework 2.0

Tip : If you need your world backup, they're located in %appdata% diretory,
copy the folder .minecraft to data in the USB Key :p

Have Fun !
